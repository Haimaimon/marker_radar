🔧 Troubleshooting:
====================================================================================================

בעיה: "Build failed"
  → בדוק requirements.txt קיים
  → בדוק שאין syntax errors

בעיה: "App crashes"
  → בדוק לוגים
  → ודא Environment Variables מוגדרים
  → בדוק TELEGRAM_BOT_TOKEN תקין

בעיה: "No notifications"
  → ודא ENABLE_TELEGRAM=true
  → בדוק Bot Token
  → בדוק Chat ID
  → ראה לוגים - יש שגיאות?

בעיה: "Too much noise"
  → ודא ENABLE_TICKER_FILTER=true
  → העלה MIN_IMPACT_SCORE
  → הפעל ENABLE_MARKET_VALIDATION
